# Demo script

Target: 2 minutes 40 seconds. Record at 1080p or higher with readable browser zoom.

## 0:00–0:20 — The problem

“AI made it cheap to start four projects in a weekend. It did not give me four sets of attention.
Most tools help me manage the pile. Fewer makes me shrink it.”

Show the sample portfolio and its four undecided projects.

## 0:20–1:25 — The cut

Open **Run the cut**. Set an honest seven-day capacity. Compare projects using the visible evidence:
signal, momentum, time ask, deadline, and next observable proof. Explain that no model invents the
winner.

Pick an active project and one maintenance obligation. Confirm.

## 1:25–1:55 — The outcome

Show:

- one active finish line;
- one maintenance slot;
- parked projects with a date and an explicit return trigger;
- the proof-shipped action asking for the next observable proof.

## 1:55–2:20 — Local-first proof

In a terminal, run the workspace scanner against a small demo folder. Open the JSON and point out
that remotes and absolute paths are absent by default. Import it into Fewer.

## 2:20–2:40 — How it was built

Show the test output and this build log. Explain the concrete Codex work and the bounded GPT-5.6
contribution. End on: “Fewer does not help you do more. It helps you finish what deserves to exist.”
